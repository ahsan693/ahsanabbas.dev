# ahsanabbas.dev

A modern, responsive personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## Overview

This project showcases my work as a developer, including:
- featured projects and case studies
- experience and education highlights
- an about section with skills and background
- a contact form connected to a backend endpoint

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Express / Node.js (for the contact API)
- Vercel Analytics

## Features

- responsive design for desktop and mobile
- animated hero and section transitions
- project cards with detailed insights
- contact form integration

## Local Development

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Project Structure

- src/components — reusable UI sections and page components
- src/assets — images, icons, and documents
- api/server.js — backend endpoint for the contact form

## Deployment

The site is designed to be deployed on Vercel.
